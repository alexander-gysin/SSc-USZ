Configuration

What this code does:
  • Defined strict quality control missingness thresholds for proteomics filtering (5% max missingness).
  • Set the clinical variable completeness threshold (variables must be 80% complete in patients to survive).
  • Established core reference dates for accurate feature engineering (Olink run date, default fallback control date).
  • Listed protected variables and explicit date columns to prevent accidental deletion during automated pruning.

Clinical Data Cleanup

Data Ingestion and Cleanup

What this code does:
  • Imported raw clinical and sample mapping Excel files and resolved ID discrepancies.
  • Engineered core analytical features including Age, Sample_Age, cohort_group, and Sex.
  • Pruned clinical variables according to the defined 80% completeness filter.
  • Enforced strict data typing and dropped manually excluded variables using the clinical_dictionary.
  • Standardized factor levels: converted ‘UN’ and ‘Unknown’ to NA.
  • Converted ‘Yes’ to 1 and ‘No’ to 0 (retained explicitly as factors to ensure mathematical compatibility with downstream models).

dict_path <- here::here("data", "clinical_dictionary.txt")
if (file.exists(dict_path)) {
  data_dict <- readr::read_tsv(dict_path, show_col_types = FALSE)
  
  # Extract classification lists
  vars_n <- data_dict %>% filter(Type == "n") %>% pull(Variable_Name)
  vars_c <- data_dict %>% filter(Type == "c") %>% pull(Variable_Name)
  vars_x <- data_dict %>% filter(Type == "x") %>% pull(Variable_Name)
  
  # Only attempt to mutate columns that survived the completeness prune
  vars_n <- intersect(vars_n, names(master_spine))
  vars_c <- intersect(vars_c, names(master_spine))
  vars_x <- intersect(vars_x, names(master_spine))
  
  # 1. Drop explicitly excluded columns
  if (length(vars_x) > 0) {
    master_spine[, (vars_x) := NULL]
  }
  
  # 2. Force Numerics (Handles Swiss decimal commas, converts text to NA safely)
  safe_numeric <- function(x) {
    if (is.numeric(x)) return(x)
    clean_x <- gsub(",", ".", trimws(as.character(x)))
    suppressWarnings(as.numeric(clean_x))
  }
  if (length(vars_n) > 0) {
    master_spine[, (vars_n) := lapply(.SD, safe_numeric), .SDcols = vars_n]
  }

  if (length(vars_c) > 0) {
    master_spine[, (vars_c) := lapply(.SD, function(x) as.factor(as.character(x))), .SDcols = vars_c]
  }

Proteomics QC, Imputation, Sync

What this code does:
  • Proteins and Samples exceeding the previously defined missingness threshold were entirely dropped.
  • Remaining NA values were imputed using a row-wise (protein-wise) median approach.
  • The master_spine clinical dataset and the Olink matrix were cross-synchronized to guarantee identical sample inclusion and ordering. 
cat("Running Missingness QC and Imputation...\n")

miss_prots <- rowMeans(is.na(olink_mat))
